<h1 id="图神经网络模型后门检测">图神经网络模型后门检测</h1>
<h2 id="后门攻击">后门攻击</h2>
<p>backdoored model会将带有特定trigger（在图数据结构中一般指特殊的子图结构）的图分类为攻击者指定的类别；一般来说，攻击者要达到的效果是，只要将trigger附着在输入图上，就会导致模型的误分类。</p>
<h2 id="目标">目标</h2>
<p>判断一个用于图分类的模型是否拥有后门，如果有，指出攻击者的目标攻击类（target class）。</p>
<h2 id="假设">假设</h2>
<p>1）黑箱模型，只能获取模型的分类置信度（softmax结果），无法得到梯度。</p>
<p>2）拥有一个干净的数据集，其中有所有类别的数据。</p>

<h2 id="方法">方法</h2>
<p>使用图上的强化学习方法。</p>
<p>相比于基于强化学习的图解释方法，这里的action space要大得多，因为解释器只需要将图中原有的边作为action space；而如果要生成触发器，action space是完全图的所有边。</p>
